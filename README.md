# 🔮 Anoma Intent Demo 🌌

Welcome to the **Anoma Intent Demo**, a mystical prototype showcasing **intent-centric Web3 experiences**.  

✨ Express your intentions and watch the system magically fulfill them with **XAN**, the native token!  

---

## ⚡ Features

- **🪄 Intent Playground:** Type or select an intent → see the magic happen!  
- **🧙 Spell Cards:** Quick actions like Swap, Private Tx, DAO Contribution, Escrow Payment.  
- **📜 Quest Log:** Track all your intents and their execution flows.  
- **🌌 Flow Visualizer:** Step-by-step view of how your intent is processed.  
- **💫 Interactive UI:** Glow effects, particles, and smooth animations.  
- **👁️ Quick Onboarding & Tooltips:** Understand what each intent and step does immediately.  

---

## 🛠️ Tech Stack

- **Frontend:** React + Tailwind CSS  
- **Animations:** Framer Motion  
- **Hosting:** Render.com  
- **Backend:** Mock API / demo data (no blockchain connection)

## 📝 Usage

1. Open the demo link  
2. Read the onboarding panel for quick guidance  
3. Click a **Spell Card** or type your intent  
4. Watch the flow execute in the **Quest Log**
